Variables Stores a value that can be changed; gives a specific output Variables are used in math as letters. Maybe someone asks a question (variable) that yields a specific answer (output). You use the equal sign to assign a value to a variable, which can be anything you want. For example, "answer = 5".
Loops A repeating circle in the code When you put your playlist on loop, it automatically goes down the list and plays the next song. You can use "for, in" to create a loop. For example, "for [variable] in range(3):" the variable would repeat 3 times.
Conditionals "If, then" statements; runs code when a certain criteria is met When scientists form hypotheses, they use "if, then" statements. Basically, "if" they perform a specific task on a variable, "then" the variable will change in a specific way. In your code, you would type "if [variable] == 'str':" and indent in the next line to write whatever code is nested in the conditional.
Functions Contains chunks of code (simplified), is reuseable, and does a specific task In math they're expressions that can be graphed. They're similar to commands. If you want text to show up in your text editor, you use the function "print()".

Well, the most complex circuit we could make that actually worked was when we connected three jumper wires with two LEDs and a resistor to a button. When we turned on the circuit and pressed the button, both the LEDs turned on, but they were very dim. Basically we connected one jumper wire to Ground and lined it up with the button. The other end of the button connected with the another jumper wire which connected with the two LEDs and resistor. Then the last jumper wire connected with the resistor to the voltage.
